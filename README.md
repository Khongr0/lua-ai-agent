# Lua Code Agent (RAG + Self-Validation)

Небольшой агент для генерации Lua-кода под задачи из workflow-систем с автоматической проверкой и доработкой.

Проект сочетает:
- поиск похожих решений (RAG)
- генерацию кода через LLM
- проверку синтаксиса
- итеративное исправление ошибок

---

##  Как это работает

1. Пользователь вводит задачу  
2. Система ищет похожие примеры в локальной базе  
3. Формируется контекст (RAG)  
4. Модель генерирует код  
5. Код проверяется через `luac`  
6. Если есть ошибки — модель пробует исправить  

Повторяется несколько итераций, пока код не станет валидным.

---

##  Структура проекта
main.py # основной скрипт (агент)
test_examples.json # база примеров для RAG

- ollama
